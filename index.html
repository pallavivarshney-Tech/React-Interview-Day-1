Day-1 React Interview Questions

Question-1

What is Virtual DOM?

The virtual Dom or(V DOM) is a programming concept where an ideal or virtual
representation of a UI is kept in memory and synced with "real dom"
by a library such as React Dom. This process is called reconciliation.

Question-2

What is SPA?

SPA stands for single page application. it is a web application or website
that interacts with the web browser by dynamically recruiting the cureent web page with new data from
web server instead of default method of browser loading entire new pages.

Question-3

Difference between functional component and class components?

Functional Components

  1. It is used to create a dumb or stateless components
  2. Functional component don't have any lifecycle methods they have hooks like UseState, UseEffect.
  3. Complete constructor structure is not required so they are easy to handle.
  4. They do not have state variables.

    Class Components

  1. Class Components is used when we are doing data operations or server calls.
  2. Class Components have lifecycle methods.
  3. Complete constructor structure is required in class components.
  4. It contains state variables.

What is JSX and why do we use it instead of js?

JS is simply a scripting language adding functionality into a webpage but JSX is an additional to the JS syntax which is a mixture
of both HTML and JavaScript.Both JS and JSX are interchangeable but JSX makes the code easier to understand for users.
